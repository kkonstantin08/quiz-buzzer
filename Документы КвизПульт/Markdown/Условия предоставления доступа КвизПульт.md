2.1. Создание и проведение викторин, подключение участников по коду, определение первого нажавшего, управление раундами, начисление баллов, отображение победителя, история игр и настройка оформления.

3.1. По окончании 30 дней платные функции ведущего становятся недоступны.

3.2. Для продолжения использования Пользователь самостоятельно оплачивает новый период. Платёжные данные не используются для автоматического повторного списания.

## 4. Изменение цены

4.1. Новая цена применяется только к будущим самостоятельным оплатам.

4.2. Оплаченный период сохраняется на прежних условиях. При наличии технической возможности об изменении направляется сообщение по email.
